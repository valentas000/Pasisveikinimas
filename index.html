Ne visi vienu metų aplankys šį puslapį. Vieni gali ryte, kiti iš vis tik po darbų 19h.
Tai pagalvojau - būtu smagu, jeigu ir pasisveikinimas priklausomai nuo dabartinio laiko kiekvienam apsilankiusiam būtu kitoks.

Štai kaip visas kodas veikia

Viskas parašyta javascript kalba

var d = new Date(); // šios dalies nepaaiškinsiu tiksliai, bet taip kuriamas datos kintamasis

var h = d.getHours(); // getHours reiškia - Gauti valandas. Dėl patogumo ir pasivadinau savo kitnamaji h kuris ir reikš mūsu valandas

Toliau susižymėjau laiką nuo 0 iki 24 valandų

var vakaras = h <= 3;
var rytas = h <= 11;
var diena = h <= 17;
var naktis = h <= 24;

Jeigu tarkim dabar yra 00 valandų, mes čia klausiam ar mūsų h kuris reliu laiku gali būti 0 yra daugiau už 3 ? Ar 2 yra daugiau už 3 ? ir ar 3 = 3 ?
na ir tolau žaidžiam su if ir else if klausimais.

if ( h = vakaras)
{
document.getElementById("labas").innerHTML = "labas vakaras";
}

else if ( h = rytas)
{
document.getElementById("labas").innerHTML = "labas rytas ! skanių pusryčių arba rytinės kavos";
}

else if ( h = diena)
{
document.getElementById("labas").innerHTML = "laba diena";
}

else if ( h = naktis)
{
document.getElementById("labas").innerHTML = "labas vakaras";
}
